# URR-Prediction-Biochar-Persulfate
This repository provides a Python-based predictive tool for unit time reaction conversion rate (URR) in biochar-activated persulfate systems, including peroxymonosulfate (PMS) and peroxydisulfate (PDS).

Method: Run the app.py file in Python, follow the prompts to enter the contaminant fingerprints, reaction conditions and biochar features, and click the "Predict" button.
