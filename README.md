# URR-Prediction-Biochar-Persulfate
This repository provides a Python-based predictive tool for unit time reaction conversion rate (URR) in biochar-activated persulfate systems, including peroxymonosulfate (PMS) and peroxydisulfate (PDS).

## Usage

1. Clone or download this repository to your local machine.

2. Ensure that the following files are located in the same directory:
   - `app.py`
   - `RF_PMS.joblib`
   - `RF_PDS.joblib`
   - `scaler_PMS.joblib`
   - `scaler_PDS.joblib`
   - `Icon.ico`

3. Install the required Python packages.
4. Launch the graphical user interface by running app.py.
